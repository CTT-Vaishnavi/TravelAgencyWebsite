## 📌 Project Overview  

The **Travel Agency Website** is a modern, responsive web application that helps users explore destinations, view travel packages, and plan trips easily.  

✨ Designed with a focus on **user experience, clean UI, and smooth navigation**.

## ⚙️ Installation / How to Run  

```bash
# Clone the repository
git clone https://github.com/ctt-vaishnavi/TravelAgencyWebsite.git

# Navigate into the project folder
cd TravelAgencyWebsite

# Open in browser
start index.html
````
